# Student Management System (Java)

A console-based Student Management System developed using Core Java. The application allows users to manage student records through a menu-driven interface.

## Features

* Add Student
* View Students
* Search Student by ID
* Update Student Details
* Delete Student
* Menu-Driven Interface

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Scanner Class

## Project Structure

Student.java

* Stores student information such as ID, name, age, course, and grade.

StudentService.java

* Handles CRUD operations for student records.

Main.java

* Provides a menu-driven interface for user interaction.

## Concepts Practiced

* Classes and Objects
* Constructors
* Encapsulation
* ArrayList Collections
* Loops and Conditional Statements
* Method Design
* CRUD Operations

## Sample Menu

1. Add Student
2. View Students
3. Search Student
4. Update Student
5. Delete Student
